# 📖 About

**HOMIES** is a browser-based community platform developed using modern web technologies with the goal of creating a simple, engaging, and user-friendly social environment.

The application is designed to allow users to create profiles, interact with other members, participate in community activities, complete daily missions, and build meaningful connections through an intuitive interface.

This project is currently under active development, and new features, improvements, and interface updates are continuously being added.

# ☁️ Firebase Notice

Earlier development versions of HOMIES used **Firebase** for authentication and cloud database services.

To protect personal development data while the project remains unfinished, the Firebase connection has been **temporarily removed** from this public repository.

This means that features requiring Firebase—such as authentication, cloud synchronization, and real-time database functionality—are currently disabled in the public version.

The Firebase integration may be restored once the project reaches a more stable stage of development.
